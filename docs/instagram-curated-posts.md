# Instagram Curated Posts — Ready to Add to Sanity

These 10 posts were selected from @guardians_of_goodness Instagram for the website's Instagram feed section. All feature cats with good rescue/adoption captions.

## How to Add to Sanity Studio

For each post below:
1. Open the Instagram URL in your browser
2. Save the main cat photo (right-click > Save Image, or screenshot)
3. In Sanity Studio, create a new **Instagram Post** document
4. Upload the image, paste the caption and URL, set visible = true, set order number
5. Publish
